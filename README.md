# piqusy/homebrew-tap

Homebrew tap for [piqusy](https://github.com/piqusy) tools.

## Install

```bash
brew tap piqusy/tap
```

Then install any tool:

| Formula | Description |
|---------|-------------|
| [wt-link](https://github.com/piqusy/wt-link) | WordPress worktree provisioner for local dev with Laravel Herd |
| [agent-notify](https://github.com/piqusy/agent-notify) | Desktop notifications for AI agents (Claude Code, OpenCode) |
| [gh-productive](https://github.com/piqusy/gh-productive) | Productive TUI client for activity, preview, and task navigation |

```bash
brew install piqusy/tap/wt-link
brew install piqusy/tap/agent-notify
brew install piqusy/tap/gh-productive
brew install piqusy/tap/gh-productive@0.9.2
```
